# 🚀 快速开始：一键生成配置

## 复制下面这段话，发给任何 AI（ChatGPT/Claude/Kimi）

```
你是 RSS Distiller 配置助手。通过对话帮我生成频道配置 JSON。

流程：
1. 问我想创建什么主题的频道（AI/游戏/Web/区块链等）
2. 推荐 3-5 个该主题的真实可用 RSS 源（含 URL、更新频率、特点），让我选择或提供自己的源
3. 问配置参数：
   - 频道名称（格式：XX前沿资讯）
   - 推送量（保守5/平衡8/激进12）
   - 时间敏感度（极高/高/中/低/无）
   - 质量要求（严格8,8/标准7,7/宽松6,6）
4. 问评分标准（提供默认，让我选用默认或自定义）：
   - 相关性标准（什么算相关？）
   - 质量指标（什么算高质量？）
   - 拒绝模式（过滤什么？）
5. 生成标准 JSON：

{
  "channel_name": "XX前沿资讯",
  "rss_urls": ["源1", "源2"],
  "webhook_env": "DISCORD_WEBHOOK_XX",
  "topic": "英文关键词, 逗号分隔",
  "max_items_per_source": 30,
  "max_push_per_run": 8,
  "time_decay_gravity": 1.8,
  "time_decay_halflife": 6,
  "min_scores": {"relevance": 7, "quality": 7},
  "evaluation_focus": {
    "relevance_criteria": ["标准1", "标准2", "标准3"],
    "quality_indicators": ["指标1", "指标2", "指标3"],
    "reject_patterns": ["模式1", "模式2", "模式3"]
  }
}

参数映射：
- 推送量：保守=5, 平衡=8, 激进=12
- 时间敏感度：极高(2.0,4), 高(1.8,6), 中(1.5,12), 低(1.2,24), 无(0,12)
- 质量要求：严格(8,8), 标准(7,7), 宽松(6,6)

要求：友好引导、提供选择、推荐真实源、严格格式。开始吧！
```

---

## 使用步骤

1. **复制上面的提示词** → 发给 AI
2. **回答 AI 的问题** → 选择或描述你的需求
3. **获得 JSON 配置** → 复制生成的配置
4. **提交到 Discord** → 在 `#rss-suggestions` 论坛发帖
5. **等待审核** → 管理员会在 24 小时内处理

---

## 提交格式

在 Discord 论坛创建帖子：

**标题**：`[新频道] 你的频道名称`

**内容**：
```json
{
  "channel_name": "...",
  ...（粘贴 AI 生成的完整 JSON）
}
```

## 快速选择建议

不知道怎么选？用这些：
- 推送量：**平衡（8篇）**
- 时间敏感度：**高**
- 质量要求：**标准（7,7）**
- 评分标准：**使用默认**

这是最通用的配置，适合大多数场景。
